¿Qué puedes hacer?

Funcionalidades principales de la API

Autenticación JWT

Regístrate o inicia sesión y obtén un token seguro para tus peticiones.

Gestión de productos

Crea, edita y consulta productos fácilmente desde la API.

Gestión de pedidos

Crea y consulta pedidos de manera segura y eficiente.

Consulta de usuarios

Accede a la información de los usuarios registrados.

Seguridad

Protección de endpoints y datos sensibles con buenas prácticas.

Rápida y escalable

Pensada para crecer y responder rápidamente a las necesidades del negocio.

Endpoints principales

Accede a los recursos clave de la API

POST /api/login Iniciar sesión
POST /api/register Registro de usuario
GET /api/products Listar productos
POST /api/orders Crear pedido

Tecnologías y Arquitectura

Esta aplicación está construida con una arquitectura modular y utiliza tecnologías modernas para ofrecer un servicio robusto y seguro.

Express.js Framework rápido y minimalista para Node.js.
Nodemailer Envío de emails para notificaciones y confirmaciones.
MongoDB Base de datos NoSQL para almacenamiento flexible.
PostgreSQL Base de datos relacional robusta y escalable.
JWT Autenticación segura basada en tokens.

¿Cómo empezar?

Sigue estos pasos para usar la API

  1. Regístrate o inicia sesión para obtener tu token JWT.
  2. Incluye el token en Authorization: Bearer <token> en las peticiones protegidas.
  3. Consulta la documentación para más detalles.
